2018-9-24日更新

（以下更新均为中秋三日经过一定的测试后推至服务器。）

①Axios静默请求队列化，页面切换时重复请求不会在请求完成前再次请求。

详阅：[Axios静默请求队列化](https://meteorocc.github.io/MyBlog/Blog/Axios/Axios%E9%9D%99%E9%BB%98%E8%AF%B7%E6%B1%82%E9%98%9F%E5%88%97%E5%8C%96/)

②路由异步获取错误时跳转至网络错误路由，并统计错误至Countly。

③全局通用状态(新消息提示、授信状态等)迁移至路由守卫中读取，业务代码不再含有这类状态的读取代码，方便后续管理。
